Akan Name Calculator
Project Description

Akan Name Calculator is a simple web application built with JavaScript, HTML, and CSS that calculates the Akan name based on the user's birthdate and gender. Inspired by Ghanaian culture, the application utilizes Zeller's Congruence Algorithm to determine the day of the week corresponding to the birthdate, then maps it to a list of traditional Akan names.

Users can input their birthdate and select their gender through an intuitive interface, and the application dynamically displays their Akan name upon submission. This project aims to provide a fun and educational tool for users interested in exploring Ghanaian traditions and naming customs.
